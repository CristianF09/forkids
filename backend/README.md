# Corcodușa - Backend

Acesta este backend-ul pentru aplicația Corcodușa, realizat cu **Node.js** și **Express**, conectat la **MongoDB** și protejat cu **JWT** pentru autentificare.

## 🔧 Cerințe preliminare

- Node.js (v16+)
- MongoDB (Atlas)
- Cont pe [Render.com](https://render.com/) pentru deploy
